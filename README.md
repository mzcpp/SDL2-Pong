# SDL2-Pong
Pong game written using SDL2 library.

Single-Player controls:
  - UP and DOWN arrows

Multi-Player controls:
  - UP and DOWN arrows for player 1
  - 'w' and 's' keys for player 2
  
Compiled with provided Makefile.
